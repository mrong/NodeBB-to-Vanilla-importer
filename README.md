# NodeBB to Vanilla importer
A tool to import NodeBB/Redis data into Vanilla forums

## Attention

**Development is in progress !!!**

## Requirements

* [Python 2.7](https://www.python.org/)
* [Paramiko](https://github.com/paramiko/paramiko)
